이진법이란?
0과 1, 두 개의 숫자만을 사용하여 수를 표현하는 진법 체계. 켜짐 / 꺼짐, 있음 / 없음 등과 같이 서로 반대되는 개념을 간단히 신호로 표기할 수 있는 진법 체계이기에 전자기기의 신호 뿐만 아니라 우리 생활의 다양한 사물 또는 현상 등을 이진법으로 표현할 수 있다.
ex) 전등이 켜진 상태 : 1, 전등이 꺼진 상태 : 0 자리에서 일어난 학생 : 1, 자리에 앉은 학생 : 0 책이 펴진 상태 : 1, 책이 덮인 상태 : 0
비트 연산이란?
비트(bit) 단위로 논리 연산을 수행할 때 사용하는 연산자. 여러 종류의 연산자가 있지만 대표적으로 AND, OR, XOR 연산자를 예로 들 수 있다.
AND 연산자(&) : 대응되는 비트가 모두 1이면 1을 반환한다.
OR 연산자(|) : 대응되는 비트 중에서 하나라도 1이면 1을 반환한다.
XOR 연산자(^) : 대응되는 비트가 서로 다르면 1을 반환한다.
